## ✅ 修复方案

### 修复1: 动态下划线占位符

**修改位置**: `my1/frontend/src/views/LearningPage.vue`

**修改前** (第92行):
```vue
:placeholder="'_'.repeat(part.length)"
```

**修改后** (第86行):
```vue
:placeholder="getPlaceholder(index)"
```

**新增方法** (第315-321行):
```javascript
// 获取动态占位符(根据当前输入长度)
const getPlaceholder = (index) => {
  const currentLength = wordInputs.value[index]?.length || 0;
  const targetLength = wordParts.value[index]?.length || 0;
  const remainingLength = Math.max(targetLength - currentLength, 0);
  return '_'.repeat(remainingLength);
};
```

**工作原理**:
1. Vue的 `v-model` 绑定 `wordInputs[index]`
2. 每次输入触发 `@input` 事件
3. `wordInputs` 数组更新触发响应式系统
4. `getPlaceholder(index)` 自动重新计算
5. 占位符实时更新

**效果演示**:
```
输入前: _____  (5个下划线,单词 "hello")
输入 h: ____   (4个下划线)
输入 he: ___   (3个下划线)
输入 hel: __   (2个下划线)
输入 hell: _   (1个下划线)
输入 hello:    (无下划线)
```

### 修复2: 只清空错误单词

**验证结果**: 代码已经正确实现,无需修改

**实现位置**: `my1/frontend/src/views/LearningPage.vue` (第428-461行)

**核心逻辑**:
```javascript
// 逐个检查每个单词是否正确
const hasErrors = [];
for (let i = 0; i < wordParts.value.length; i++) {
  const input = wordInputs.value[i].trim().toLowerCase();
  const expected = wordParts.value[i].toLowerCase();
  if (input !== expected) {
    hasErrors.push(i);
    wordErrors.value[i] = true;
  } else {
    wordErrors.value[i] = false;
  }
}

// 只清空错误的单词
hasErrors.forEach(index => {
  wordInputs.value[index] = '';
  wordErrors.value[index] = false;
});

// 聚焦到第一个错误的输入框
if (hasErrors.length > 0) {
  currentWordIndex.value = hasErrors[0];
  nextTick(() => {
    wordInputRefs.value[hasErrors[0]]?.focus();
  });
}
```

**功能特点**:
- ✅ 逐个比对每个单词
- ✅ 只标记错误的单词
- ✅ 只清空错误的单词
- ✅ 保留正确的单词
- ✅ 显示错误数量提示
- ✅ 自动聚焦到第一个错误的输入框
- ✅ 错误单词有红色下划线动画

## 📊 边界情况处理

### getPlaceholder 方法边界情况
| 情况 | currentLength | targetLength | remainingLength | 结果 |
|------|---------------|--------------|-----------------|------|
| 空输入 | 0 | 5 | 5 | `_____` |
| 部分输入 | 3 | 5 | 2 | `__` |
| 完整输入 | 5 | 5 | 0 | `` |
| 超长输入 | 7 | 5 | 0 | `` |
| 未定义 | 0 (默认) | 5 | 5 | `_____` |

**处理方式**:
- `?.length || 0` 处理 undefined/null
- `Math.max(..., 0)` 确保非负数
- `'_'.repeat(0)` 返回空字符串

## 🔧 技术实现细节

### Vue响应式系统
```javascript
// v-model 双向绑定
v-model="wordInputs[index]"

// 自动触发更新
@input="handleWordInput(index)"

// 计算属性自动重新计算
:placeholder="getPlaceholder(index)"
```

### 错误处理流程
```
用户输入完成
    ↓
按空格提交 (handleWordComplete)
    ↓
所有单词输入完成 (handleMultiWordSubmit)
    ↓
后端验证答案 (learningService.checkAnswer)
    ↓
逐个比对单词 (for loop)
    ↓
标记错误单词 (wordErrors[i] = true)
    ↓
显示错误提示 + 播放音频
    ↓
1.5秒后
    ↓
只清空错误的单词 (hasErrors.forEach)
    ↓
聚焦到第一个错误的输入框
```
