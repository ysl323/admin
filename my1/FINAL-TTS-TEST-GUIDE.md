## 🚀 快速测试流程

### 步骤 1：启动服务

```bash
# 检查服务状态
cd my1
check-services.bat
```

确保看到：
- ✅ Backend: http://localhost:3000 (running)
- ✅ Frontend: http://localhost:5173 (running)

### 步骤 2：访问配置页面

1. 打开浏览器：http://localhost:5173
2. 登录管理员账号：
   - 用户名：`admin`
   - 密码：`admin123`
3. 点击右上角蓝色按钮"进入后台"
4. 左侧菜单选择"配置管理"

### 步骤 3：测试简单模式

1. 选择"火山引擎 TTS"标签页
2. 点击"简单模式"单选按钮
3. 填写配置：
   ```
   AppID: 8594935941
   Access Token: sRWjJ3dvQhY4ZnyqYgdj331lQ2WkNPL
   Secret Key: hLY8jzW6WNguVHyZovHBPMkXt_4ZLdFR
   ```
4. 点击"保存配置"
5. 等待提示"配置保存成功"

### 步骤 4：测试配置

1. 点击"测试配置"按钮
2. 在弹出的对话框中：
   - 查看默认文本：`Hello, this is a test.`
   - 或输入自定义文本：`Excuse me!`
3. 点击"开始测试"
4. 观察测试结果

**预期结果**：
- ✅ 对话框正常弹出
- ✅ 可以输入自定义文本
- ✅ 点击测试后显示结果
- ⚠️ 如果显示 401 错误，这是火山引擎 API 的认证问题，不影响配置功能

### 步骤 5：测试复杂模式

1. 点击"复杂模式"单选按钮
2. 观察高级配置项是否显示：
   - 接口地址
   - 默认音色
   - 语言
3. 修改音色为"英文男声 (BV701_streaming)"
4. 点击"保存配置"
5. 刷新页面（F5）
6. 重新登录并进入配置管理
7. 检查配置是否保持

### 步骤 6：测试配置脱敏

1. 观察 Access Token 的显示：
   - 应该显示为：`sRWj****NPL`（前4位+****+后4位）
2. 观察 Secret Key 的显示：
   - 应该显示为：`hLY8****dFR`
3. AppID 应该完整显示：`8594935941`

### 步骤 7：测试模式切换

1. 当前在复杂模式，点击"简单模式"
2. 观察高级配置项是否隐藏
3. 点击"复杂模式"
4. 观察高级配置项是否重新显示
5. 配置值应该保持不变


### 问题 2：登录失败

**解决方法**：
- 确认用户名：`admin`
- 确认密码：`admin123`
- 检查后端服务是否正常

**解决方法**：
1. 打开浏览器控制台（F12）
2. 查看 Network 标签的请求详情

**说明**：
- 这是火山引擎 API 的认证问题
