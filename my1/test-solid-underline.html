✅ 实线下划线修复测试

1. 单个单词输入（短单词）

目标单词长度: 3个字母 (cat)

2. 单个单词输入（中等长度）

目标单词长度: 5个字母 (hello)

3. 单个单词输入（长单词）

目标单词长度: 9个字母 (beautiful)

4. 多单词输入

目标短语: "good morning" (两个单词)

5. 错误状态（红色下划线）

错误输入会显示红色下划线并抖动

6. 禁用状态

答对后输入框会被禁用

✅ 修复说明

  • 使用 CSS border-bottom：每个输入框下方显示一条连续的实线
  • 不再使用下划线字符：移除了所有 "_" 字符的占位符
  • 动态宽度：根据目标单词长度自动调整输入框宽度
  • 蓝色实线：默认显示 #409eff 颜色的实线
  • 聚焦效果：聚焦时实线变为浅蓝色 #66b1ff
  • 错误状态：错误时实线变为红色 #f56c6c
