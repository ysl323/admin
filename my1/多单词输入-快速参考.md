## 📖 使用方法

### 单个单词 (hello)
```
1. 输入 "hello"
2. 按空格或回车
3. 提交答案
```

### 多个单词 (hello world)
```
1. 输入 "hello" → 按空格
   ✅ 正确 → 自动跳到下一个
   ❌ 错误 → 清空、播放音频、重新输入

2. 输入 "world" → 按空格
   ✅ 正确 → 提交完整答案
   ❌ 错误 → 清空、播放音频、重新输入
```

## 🎨 视觉效果

### 正常
```
[_____] [_____]
  ↑ 蓝色下划线
```

### 输入中
```
[hello] [_____]
  ↑ 绿色    ↑ 蓝色高亮
```

### 错误
```
[helo] [_____]
  ↑ 红色 + 抖动
```
