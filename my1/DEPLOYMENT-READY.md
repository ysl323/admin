## 🔧 如果遇到问题

### 部署失败
1. 检查网络连接
2. 确认服务器可访问
3. 查看错误信息
4. 参考 `DEPLOY-NOW.md` 的故障排查部分

### 没有 SSH 工具
安装 OpenSSH 客户端：
```powershell
# 以管理员身份运行 PowerShell
Add-WindowsCapability -Online -Name OpenSSH.Client~~~~0.0.1.0
```

### 需要手动部署
查看 `DEPLOY-NOW.md` 中的"方法 2: 使用 FTP 工具"部分
