# 火山引擎 TTS 配置完整指南

## 📋 功能概述

本系统支持火山引擎和谷歌两种 TTS 提供商，其中火山引擎 TTS 提供了两种配置模式：

- **简单模式**：只需填写核心密钥（AppID、Access Token、Secret Key），适合快速配置
- **复杂模式**：可自定义接口地址、音色、语言等高级参数，适合精细化配置

## 🔧 功能特性

### 1. 模式切换
- 在配置页面可以随时切换简单模式和复杂模式
- 切换到简单模式时，高级配置项会使用默认值
- 切换到复杂模式时，可以自定义所有配置项

### 2. 配置保存
- 所有敏感信息（Access Token、Secret Key）都会加密存储
- 配置保存后会立即生效
- 支持分别保存火山引擎和谷歌的配置

### 3. 配置测试
- 点击"测试配置"按钮会弹出测试对话框
- 可以输入自定义测试文本（默认：Hello, this is a test.）
- 测试会实际调用 TTS API 生成语音
- 测试成功会显示成功消息
- 测试失败会显示详细的错误信息

### 4. 配置脱敏
- 加载配置时，敏感信息会被脱敏显示（只显示前4位和后4位）
- 例如：sRWjJ3dvQhY4ZnyqYgdj331lQ2WkNPL 显示为 sRWj****NPL

## 🚀 快速开始

### 步骤 1：访问配置页面

1. 打开浏览器访问 http://localhost:5173
2. 使用管理员账号登录：
   - 用户名：`admin`
   - 密码：`admin123`
3. 点击右上角蓝色的"进入后台"按钮
4. 在左侧菜单选择"配置管理"

### 步骤 2：配置火山引擎 TTS（简单模式）

1. 确保选中"火山引擎 TTS"标签页
2. 选择"简单模式"单选按钮
3. 填写核心配置：
   ```
   AppID: 8594935941
   Access Token: sRWjJ3dvQhY4ZnyqYgdj331lQ2WkNPL
   Secret Key: hLY8jzW6WNguVHyZovHBPMkXt_4ZLdFR
   ```
4. 点击"保存配置"按钮
5. 等待提示"配置保存成功"

### 步骤 3：测试配置

1. 点击"测试配置"按钮
2. 在弹出的对话框中：
   - 查看默认测试文本：`Hello, this is a test.`
   - 或者输入自定义测试文本，例如：`Excuse me!`
3. 点击"开始测试"按钮
4. 等待测试结果：
   - ✅ 成功：显示"火山引擎 TTS 配置测试成功"
   - ❌ 失败：显示具体的错误信息

## 🐛 常见问题排查

### 问题 1：测试失败，提示"网络请求失败"

**可能原因**：
- 网络连接问题
- 接口地址错误
- 防火墙阻止

**解决方法**：
1. 检查网络连接
2. 确认接口地址：`https://openspeech.bytedance.com/api/v1/tts`
3. 检查防火墙设置

### 问题 2：测试失败，提示"API 调用失败"

**可能原因**：
- Access Token 或 Secret Key 错误
- AppID 错误
- 账号权限不足

**解决方法**：
1. 核对火山引擎控制台的配置信息
2. 确认账号有 TTS 服务权限
3. 重新生成 Access Token

### 问题 3：配置保存后刷新页面丢失

**可能原因**：
- 数据库连接问题
- 后端服务异常

**解决方法**：
1. 检查后端服务是否正常运行
2. 查看后端日志：`my1/backend/logs/`
3. 重启后端服务

### 问题 4：模式切换后配置丢失

**可能原因**：
- 前端状态管理问题

**解决方法**：
1. 切换模式后重新填写配置
2. 保存配置后再切换模式
3. 刷新页面重新加载

## 🔐 安全说明

1. **配置加密**：所有敏感信息（Access Token、Secret Key）在数据库中都是加密存储的
2. **配置脱敏**：前端显示时会自动脱敏，只显示部分字符
3. **权限控制**：只有管理员账号才能访问配置管理页面
4. **HTTPS 建议**：生产环境建议使用 HTTPS 传输配置信息
