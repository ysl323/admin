## 常见问题

### Q1: 导入失败，提示"JSON 格式错误"

**解决方案：**
1. 检查 JSON 格式是否正确
2. 确保所有字符串使用双引号
3. 确保没有多余的逗号
4. 使用在线 JSON 验证工具检查

### Q2: TTS 测试失败

**解决方案：**
1. 检查配置信息是否完整
2. 检查 Access Token 和 Secret Key 是否正确
3. 检查网络连接
4. 查看浏览器控制台的错误信息

### Q3: 学习页面音频不播放

**解决方案：**
1. 当前使用浏览器自带的 Speech Synthesis API
2. 确保浏览器支持该功能
3. 检查浏览器音量设置
4. 后续会集成火山引擎 TTS

### Q4: 导入的单词顺序不对

**解决方案：**
1. 检查 question 字段的值
2. 系统会自动按 question 从小到大排序
3. 确保 question 值连续且唯一
