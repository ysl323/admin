## 📝 已修复的问题

### 问题1: 数据库缺少 `is_super_admin` 列
**原因**: User 模型定义了 `isSuperAdmin` 字段，但数据库表中没有这个列

**解决方案**: 执行 SQL 添加列
```sql
ALTER TABLE users ADD COLUMN is_super_admin BOOLEAN DEFAULT 0;
```

### 问题2: admin 密码验证失败
**原因**: 之前的密码哈希值有问题

**解决方案**: 使用 bcrypt 重新生成密码哈希
```javascript
const hashedPassword = await bcrypt.hash('admin123', 10);
```

### 问题3: admin 权限不完整
**原因**: admin 用户没有超级管理员权限

**解决方案**: 更新用户权限
```sql
UPDATE users SET is_admin = 1, is_super_admin = 1 WHERE username = 'admin';
```
