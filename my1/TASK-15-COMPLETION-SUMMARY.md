## 技术实现细节

### 前端技术栈
- Vue 3 Composition API
- Element Plus UI 组件库
- Vue Router 路由管理
- Axios HTTP 客户端

### 核心组件
- `LearningPage.vue`：学习页面主组件
- `NavBar.vue`：顶部导航栏
- `AudioManager.js`：音频管理工具类

### API 集成
- `GET /api/lessons/:id/words`：获取课程单词列表
- `POST /api/words/check-answer`：检查答案正确性
- 返回：正确/错误状态、正确答案、最长公共前缀

### 状态管理
- `loading`：加载状态
- `words`：单词列表
- `currentIndex`：当前题号
- `userAnswer`：用户输入
- `showAnswer`：是否显示答案
- `isChecking`：是否正在检查答案
- `isPlaying`：是否正在播放音频
- `showCompleteDialog`：是否显示完成对话框
- `feedback`：反馈信息（显示、类型、消息）

### 响应式设计
- 使用 CSS 媒体查询适配移动设备
- 移动端布局垂直排列
- 按钮宽度 100%
- 字体大小自适应
