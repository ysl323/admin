# 前端功能测试指南

## 当前已完成的功能

### ✅ 任务 13: 前端登录和注册页面

#### 1. 注册页面 (http://localhost:5173/register)
- 用户名输入（3-20个字符）
- 密码输入（至少6个字符）
- 确认密码
- 表单验证
- 注册成功后跳转到登录页
- 显示试用期提示（3天）

#### 2. 登录页面 (http://localhost:5173/login)
- 用户名输入
- 密码输入
- 表单验证
- 登录成功后跳转到分类首页
- 错误提示

#### 3. 导航栏组件
- 显示用户名
- 用户下拉菜单
- 退出登录功能（带确认对话框）
- 管理后台入口（仅管理员可见）

## 测试步骤

### 1. 测试注册功能
1. 访问 http://localhost:5173/register
2. 输入用户名：testuser
3. 输入密码：test123
4. 确认密码：test123
5. 点击"注册"按钮
6. 应该看到"注册成功！请登录"提示
7. 自动跳转到登录页

### 2. 测试登录功能
1. 访问 http://localhost:5173/login
2. 输入用户名：testuser
3. 输入密码：test123
4. 点击"登录"按钮
5. 应该看到"登录成功"提示
6. 自动跳转到分类首页

### 3. 测试退出登录
1. 在分类首页，点击右上角的用户图标
2. 选择"退出登录"
3. 确认退出
4. 应该看到"已退出登录"提示
5. 自动跳转到登录页

### 4. 测试管理员登录
1. 访问 http://localhost:5173/login
2. 输入用户名：admin
3. 输入密码：admin123
4. 点击"登录"按钮
5. 登录成功后，用户下拉菜单应该显示"管理后台"选项

## 已实现的 API 服务

- ✅ api.js - Axios 实例和拦截器
- ✅ auth.js - 认证服务（登录、注册、退出、检查认证）
- ✅ learning.js - 学习服务（分类、课程、单词、答案检查）
- ✅ admin.js - 管理员服务（用户管理、内容管理、TTS配置）
- ✅ tts.js - TTS 服务（语音合成）

## 已实现的工具类

- ✅ AudioManager.js - 音频管理器（播放、预加载、缓存）

## 路由配置

- ✅ / → 重定向到 /categories
- ✅ /login → 登录页
- ✅ /register → 注册页
- ✅ /categories → 分类首页（需要登录）
- ✅ /categories/:id/lessons → 课程列表（需要登录）
- ✅ /lessons/:id/learning → 学习页面（需要登录）
- ✅ /admin → 管理后台（需要登录 + 管理员权限）
  - /admin/users → 用户管理
  - /admin/content → 内容管理
  - /admin/config → 配置管理
