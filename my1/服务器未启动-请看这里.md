# 🔴 服务器未启动

## 问题

你看到的浏览器错误是因为：**前端和后端服务器都没有运行**

## 解决方法

### 方法1：一键启动（推荐）

双击运行这个文件：

```
start-all.bat
```

这会同时启动前端和后端服务器。

### 方法2：手动启动

#### 启动后端（端口 3000）

```bash
cd my1/backend
npm start
```

#### 启动前端（端口 5173）

```bash
cd my1/frontend
npm run dev
```

## 如何确认服务器已启动

### 后端服务器

访问：http://localhost:3000/health

应该看到：`{"status":"ok"}`

### 前端服务器

访问：http://localhost:5173

应该看到登录页面

## 你看到的错误

```
Third-party cookie will be blocked
Mixed Content warnings
connection failure
```

这些都是 **360浏览器的错误页面**，不是我们的应用。

这些错误出现是因为：
1. 前端服务器（5173端口）没有运行
2. 浏览器无法连接到 localhost:5173
3. 360浏览器显示了它自己的错误页面
