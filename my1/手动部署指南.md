### 方法2: 手动分步执行

如果脚本失败,按以下步骤手动操作:

#### 1. 打包文件
在 my1 目录执行:
```powershell
# 创建时间戳
$timestamp = Get-Date -Format "yyyyMMdd_HHmmss"
$deployDir = "deploy-$timestamp"

# 创建目录
New-Item -ItemType Directory -Path "$deployDir\backend\src" -Force
New-Item -ItemType Directory -Path "$deployDir\frontend\dist" -Force

# 复制文件
Copy-Item -Recurse backend\src\* "$deployDir\backend\src\"
Copy-Item backend\package.json "$deployDir\backend\"
Copy-Item backend\package-lock.json "$deployDir\backend\" -ErrorAction SilentlyContinue
Copy-Item -Recurse frontend\dist\* "$deployDir\frontend\dist\"
Copy-Item .env "$deployDir\" -ErrorAction SilentlyContinue
Copy-Item nginx-english-learning.conf "$deployDir\" -ErrorAction SilentlyContinue

# 压缩
Compress-Archive -Path $deployDir -DestinationPath "$deployDir.zip" -Force
```

#### 2. 上传到服务器
```powershell
scp deploy-*.zip root@47.97.185.117:/root/
# 密码: Admin88868
```

#### 3. 登录服务器并部署
```powershell
ssh root@47.97.185.117
# 密码: Admin88868
```

在服务器上执行:
```bash
cd /root
bash deploy-server.sh
```
