## 🎯 本次修复内容

### 1. TTS API 404错误修复
- **问题**：前端调用 POST `/api/tts/speak` 返回404错误
- **原因**：后端只定义了 GET 路由，没有 POST 路由
- **修复**：在 `backend/src/routes/tts.js` 添加 POST `/speak` 路由
- **影响**：TTS播放功能现在应该完全正常工作

### 2. 课程列表显示修复
- **问题**：课程列表显示"未知分类"和"0个单词"
- **原因**：
  - 后端没有返回 `wordCount` 字段
  - 前端使用错误的字段名
- **修复**：
  - 后端添加 `wordCount` 字段
  - 前端修正为 `category.name`
- **影响**：课程列表现在应该正确显示分类名称和单词数量

### 3. 学习页面返回按钮
- **问题**：学习页面缺少返回按钮
- **修复**：添加"返回课程"按钮
- **影响**：用户可以方便地返回课程列表

## ⚠️ 常见问题

### Q1: TTS还是返回404错误
**A**: 
1. 确认后端服务已重启（修改代码后需要重启）
2. 检查 `backend/src/routes/tts.js` 是否有 POST `/speak` 路由
3. 清除浏览器缓存后重试

### Q2: 课程列表还是显示"未知分类"
**A**:
1. 确认后端服务已重启
2. 检查数据库中是否有分类数据
3. 查看后端日志是否有错误

### Q3: 没有声音
**A**:
1. 检查浏览器是否允许音频播放
2. 检查音量设置
3. 查看网络请求是否成功
4. 检查火山引擎TTS配置

### Q4: 返回按钮不显示
**A**:
1. 确认前端已重新编译（Vite会自动热更新）
2. 刷新浏览器页面
3. 清除浏览器缓存
