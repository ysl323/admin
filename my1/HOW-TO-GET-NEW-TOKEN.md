# 如何获取新的火山引擎 TTS Access Token

## 问题说明

当前使用的 Access Token 已过期,导致 API 调用返回 3001 错误 "access denied"。

## 解决步骤

### 步骤 1: 登录火山引擎控制台

访问: https://console.volcengine.com/

### 步骤 2: 进入语音合成服务

1. 在控制台左侧菜单中找到 "音频技术"
2. 点击 "语音合成" 或 "豆包语音"

### 步骤 3: 找到您的应用

查找 AppID 为 `2128862431` 的应用

### 步骤 4: 重新生成 Access Token

1. 点击应用进入详情页
2. 找到 "Access Token" 或 "访问令牌" 部分
3. 点击 "重新生成" 或 "刷新" 按钮
4. 复制新生成的 Access Token

### 步骤 5: 更新系统配置

有两种方式更新配置:

#### 方式 A: 使用管理后台 (推荐)

1. 登录系统管理后台
2. 进入 "配置管理" → "TTS 配置"
3. 选择 "火山引擎"
4. 更新 Access Token
5. 点击 "保存" 并 "测试配置"
