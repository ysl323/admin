## ⚠️ 已知问题

### 1. 火山引擎 TTS API 返回 401

**问题描述**：
- 后端调用火山引擎 TTS API 时返回 401 Unauthorized
- 可能原因：签名生成方式不正确

**影响范围**：
- 实际 TTS 测试功能无法使用
- 不影响配置保存和加载功能

**解决方案**：
1. 检查火山引擎官方文档的签名生成方式
2. 验证 AppID、Access Token、Secret Key 是否正确
3. 检查请求头格式是否符合要求
4. 可能需要使用火山引擎官方 SDK

### 2. 配置脱敏可能影响重新保存

**问题描述**：
- 配置加载时会脱敏显示（sRWj****NPL）
- 如果用户不修改直接保存，可能会保存脱敏后的值

**解决方案**：
- 前端需要判断：如果值是脱敏格式，保存时不发送该字段
- 或者后端判断：如果收到脱敏格式的值，使用原有值
