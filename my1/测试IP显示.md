# 测试IP显示

## 前端检查步骤

### 1. 打开浏览器开发者工具

按 `F12` 打开开发者工具

### 2. 查看Network标签

1. 刷新用户管理页面
2. 找到 `/api/admin/users` 请求
3. 查看Response，确认返回的数据中有 `lastLoginIp` 字段

### 3. 查看Console标签

看看有没有JavaScript错误

### 4. 检查前端代码

前端表格列定义：

```vue
<el-table-column prop="lastLoginIp" label="最后登录IP" width="140" />
```

这个应该能正常显示 `lastLoginIp` 字段。

## 可能的问题

1. **前端缓存** - 浏览器缓存了旧的JS文件
2. **后端未重启** - 修改后端代码后没有重启
3. **数据未刷新** - 页面加载的是旧数据

## 解决方法

### 方法1：强制刷新浏览器

```
Ctrl + Shift + R  (强制刷新，清除缓存)
或
Ctrl + F5
```

### 方法2：重启后端

```bash
# 停止后端（Ctrl+C）
# 然后重新启动
cd my1/backend
npm start
```

### 方法3：清除浏览器缓存

1. 按 `Ctrl + Shift + Delete`
2. 选择"缓存的图片和文件"
3. 点击"清除数据"
4. 刷新页面
