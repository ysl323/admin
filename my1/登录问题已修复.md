## 如果还是不能跳转

### 检查1: 清除浏览器缓存
1. 按 `Ctrl + Shift + Delete`
2. 清除缓存和Cookie
3. 刷新页面重新登录

### 检查2: 查看浏览器控制台
1. 按 `F12` 打开开发者工具
2. 切换到 Console 标签
3. 查看是否有错误信息
4. 切换到 Network 标签
5. 重新登录，查看 `/api/auth/login` 请求的响应

### 检查3: 验证Session
登录成功后，后端会创建session:
```javascript
req.session.userId = result.user.id;
req.session.isAdmin = result.user.isAdmin;
```

如果session没有正确保存，可能是CORS或cookie设置问题。
