# BUG #10: 下划线完全不显示 - 紧急修复

## 修复时间

## 严重问题发现

用户提供截图显示：**下划线完全不显示！**

截图显示：
- 输入框显示 "lkdsfk"  
- 输入框下方有一条蓝色的线
- **但是没有下划线字符（_____）**

## 根本原因

我之前的修复（BUG #8）添加了：
```css
.underline-placeholder {
  width: 100%;           /* 问题！ */
  overflow: hidden;      /* 问题！ */
}
```

**这导致**：
- 下划线placeholder被设置为和输入框一样宽
- 但是`overflow: hidden`把下划线字符裁剪掉了
- 所以用户看不到下划线字符

## 正确的修复

**删除错误的CSS**：
- 删除 `width: 100%`
- 删除 `overflow: hidden`

**添加正确的CSS**：
```css
.underline-placeholder {
  color: #409eff;
  font-size: 32px;
  font-weight: 500;
  letter-spacing: 2px;
  pointer-events: none;
  user-select: none;
  white-space: nowrap;
  text-align: center;
  min-height: 40px;      /* 新增：确保有高度 */
  line-height: 40px;     /* 新增：垂直居中 */
}
```

## 修复原理

1. **不限制宽度**：让下划线自然显示，宽度由内容决定
2. **设置最小高度**：`min-height: 40px` 确保下划线有足够空间显示
3. **设置行高**：`line-height: 40px` 让下划线垂直居中
4. **保持gap: 0px**：下划线紧贴输入框下方
