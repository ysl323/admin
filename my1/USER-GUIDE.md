# 编程英语学习系统 - 使用指南

## 🎯 系统简介

这是一个专为程序员设计的英语学习系统，帮助你掌握编程相关的英语词汇和表达。

### 主要功能
- 📚 分类学习：按主题分类的课程体系
- 🎵 语音播放：火山引擎TTS高质量发音
- ✍️ 互动练习：输入答案即时反馈
- 📊 进度跟踪：记录学习进度
- 👨‍💼 后台管理：管理员可管理内容

---

## 📖 用户功能

### 注册和登录

#### 注册新用户
1. 点击"注册"按钮
2. 填写用户信息：
   - 用户名（唯一）
   - 密码（至少6位）
   - 确认密码
3. 点击"注册"
4. 注册成功后自动登录

#### 登录系统
1. 输入用户名和密码
2. 点击"登录"
3. 登录成功后进入分类页面

---

### 学习流程

#### 1. 选择分类
- 在首页查看所有学习分类
- 每个分类显示课程数量
- 点击分类卡片进入课程列表

#### 2. 选择课程
- 查看该分类下的所有课程
- 课程按编号排序
- 点击课程进入学习页面

#### 3. 学习单词
- 查看中文提示
- 输入英文单词
- 按空格键或回车键提交答案
- 系统自动检查答案

#### 4. 答案反馈
- **正确**：显示绿色提示，1秒后自动跳转下一题
- **错误**：显示红色提示，显示正确答案，1.5秒后隐藏答案
- 错误时保留最长公共前缀，方便继续输入

#### 5. 控制按钮
- **上一题**：返回上一个单词
- **播放当前**：播放当前单词的语音
- **显示答案**：直接显示正确答案
- **重新本题**：清空输入，重新开始本题
- **下一题**：跳转到下一个单词

#### 6. 完成学习
- 完成所有单词后显示完成对话框
- 可以选择：
  - 返回课程列表
  - 重新学习

---

### 语音播放

#### 自动播放
- 进入学习页面时自动播放2遍
- 切换单词时自动播放2遍
- 答对后跳转下一题时自动播放2遍

#### 手动播放
- 点击"播放当前"按钮播放1遍
- 使用火山引擎TTS，发音清晰标准

#### 降级方案
- 如果火山引擎TTS不可用
- 自动降级到浏览器语音合成
- 保证基本功能可用

---

### 进度跟踪

#### 访问权限
- 每个用户有访问天数限制
- 天数每天凌晨2点自动递减
- 剩余3天或更少时显示提醒
- 天数为0时无法访问学习内容

#### 学习记录
- 系统记录学习进度
- 显示当前题号和总题数
- 显示进度百分比

## 👨‍💼 管理员功能

### 访问后台

1. 以管理员身份登录
2. 点击右上角头像
3. 选择"后台管理"
4. 进入管理后台

---

### 用户管理

#### 查看用户列表
- 显示所有注册用户
- 显示用户名、角色、访问天数
- 支持搜索用户

#### 编辑用户
- 修改用户角色（普通用户/管理员）
- 修改访问天数
- 重置密码

#### 删除用户
- 删除指定用户
- 删除后无法恢复

---

### 内容管理

#### 分类管理
1. 查看所有分类
2. 显示每个分类的课程数量
3. 操作：
   - **新增分类**：创建新的学习分类
   - **编辑分类**：修改分类名称
   - **删除分类**：删除分类（会级联删除课程和单词）

#### 课程管理
1. 查看所有课程
2. 按分类筛选课程
3. 显示每个课程的单词数量
4. 操作：
   - **新增课程**：在指定分类下创建课程
   - **编辑课程**：修改课程编号
   - **删除课程**：删除课程（会级联删除单词）

#### 单词管理
1. 查看所有单词
2. 按课程筛选单词
3. 搜索英文或中文
4. 操作：
   - **新增单词**：在指定课程下添加单词
   - **编辑单词**：修改英文、中文或所属课程
   - **删除单词**：删除指定单词

---

### 一键导入

#### 功能说明
- 批量导入课程数据
- 支持JSON格式
- 自动创建分类、课程和单词

#### 使用步骤
1. 点击"一键导入课程"按钮
2. 输入分类名称（例如：新概念英语第一册）
3. 粘贴JSON数据
4. 点击"开始导入"
5. 等待导入完成

#### JSON格式
```json
[
  {
    "lesson": 1,
    "question": 1,
    "english": "Excuse me!",
    "chinese": "打扰一下！"
  },
  {
    "lesson": 1,
    "question": 2,
    "english": "Yes?",
    "chinese": "什么事？"
  }
]
```

**字段说明**：
- `lesson`（可选）：课时号，有则按课时分组
- `question`（必填）：序号
- `english`（必填）：英文单词或句子
- `chinese`（必填）：中文翻译

#### 导入结果
- 显示导入统计：
  - 分类名称
  - 创建的课程数量
  - 创建的单词数量
  - 是否按课时分组

## 🔧 常见问题

### 1. 无法登录
**问题**：输入正确的用户名和密码但无法登录

**解决方案**：
- 检查用户名和密码是否正确
- 检查网络连接
- 清除浏览器缓存
- 检查后端服务是否运行

### 2. 课程数量显示为0
**问题**：分类页面显示"0 个课程"

**解决方案**：
- 刷新页面
- 检查数据库中是否有课程数据
- 查看浏览器控制台错误
- 查看后端日志

### 3. TTS没有声音
**问题**：点击播放按钮没有声音

**解决方案**：
- 检查浏览器是否允许自动播放
- 检查音量设置
- 检查火山引擎TTS配置
- 查看浏览器控制台错误
- 系统会自动降级到浏览器语音合成

### 4. 导入失败
**问题**：一键导入时提示失败

**解决方案**：
- 检查JSON格式是否正确
- 确保必填字段完整
- 检查分类名称是否已存在
- 查看后端日志错误信息

### 5. 访问权限到期
**问题**：提示"访问权限已到期"

**解决方案**：
- 联系管理员续期
- 管理员在用户管理中修改访问天数

## 💡 使用技巧

### 学习技巧
1. **每天坚持**：每天学习一定数量的单词
2. **重复练习**：对错误的单词多次练习
3. **听力训练**：多听语音，培养语感
4. **实际应用**：在编程中使用学到的词汇

### 管理技巧
1. **分类规划**：合理规划学习分类
2. **课程编号**：使用有意义的课程编号
3. **批量导入**：使用一键导入提高效率
4. **定期备份**：定期备份数据库
