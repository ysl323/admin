## 📋 如果一键部署失败

### 情况 1: 没有 SSH 工具

**解决方案**: 安装 OpenSSH 客户端

以管理员身份运行 PowerShell：
```powershell
Add-WindowsCapability -Online -Name OpenSSH.Client~~~~0.0.1.0
```

然后重新运行 `一键部署.bat`

### 情况 2: 网络问题

**解决方案**: 使用 FTP 手动上传

1. 下载 [FileZilla](https://filezilla-project.org/)
2. 连接到服务器：
   - 主机: `sftp://47.97.185.117`
   - 用户: `root`
   - 密码: `Admin88868`
   - 端口: `22`
3. 上传 `deploy-package.zip` 到 `/root/`
4. 上传 `server-setup.sh` 到 `/root/`
5. 使用 SSH 工具连接服务器运行：
   ```bash
   cd /root
   chmod +x server-setup.sh
   ./server-setup.sh
   ```
