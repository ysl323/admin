## 🔍 调试技巧

### 浏览器控制台
按 F12 打开开发者工具：
- **Console**：查看JavaScript错误和日志
- **Network**：查看API请求和响应
- **Application**：查看LocalStorage中的token

### 后端日志
查看后端终端输出：
- 绿色 `info`：正常日志
- 黄色 `warn`：警告信息
- 红色 `error`：错误信息

### 常见问题

**问题1：课程数量显示为0**
- 检查后端API响应是否包含 `lessonCount` 字段
- 检查数据库中是否有课程数据

**问题2：TTS没有声音**
- 检查火山引擎TTS配置
- 检查网络请求是否成功
- 检查浏览器是否允许自动播放音频

**问题3：后台管理页面空白**
- 检查是否以管理员身份登录
- 检查后端API是否正常
- 检查浏览器控制台错误

**问题4：一键导入失败**
- 检查JSON格式是否正确
- 检查必填字段是否完整
- 检查后端日志错误信息

## 📊 API测试

### 使用curl测试

**1. 获取分类列表（带课程数量）**
```bash
curl -X GET "http://localhost:3000/api/learning/categories" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

**2. 测试TTS播放**
```bash
curl -X POST "http://localhost:3000/api/tts/speak" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d "{\"text\":\"Hello\"}" \
  --output test-audio.mp3
```

**3. 获取后台分类列表**
```bash
curl -X GET "http://localhost:3000/api/admin/categories" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### 获取Token

1. 登录获取token：
```bash
curl -X POST "http://localhost:3000/api/auth/login" \
  -H "Content-Type: application/json" \
  -d "{\"username\":\"admin\",\"password\":\"admin123\"}"
```

2. 从响应中复制token
3. 替换上面命令中的 `YOUR_TOKEN`
