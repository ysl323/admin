### 问题1：SSH连接失败

- 服务器IP是否正确：47.97.185.117
- 用户名是否正确：root
- 密码是否正确：MyEnglish2025!

### 问题2：PM2命令找不到
```bash

pm2 list
```

### 问题3：前端恢复后仍然500
```bash
# 清除PM2缓存
pm2 flush
pm2 restart my1-backend

# 清除浏览器缓存
# 按 Ctrl + F5 强制刷新
```

### 问题4：数据库问题
```bash
cd /root/my1-english-learning/backend
ls -lh database.sqlite

# 如果数据库损坏
pm2 stop my1-backend
rm database.sqlite
# 系统会自动重新创建
pm2 start my1-backend
```
