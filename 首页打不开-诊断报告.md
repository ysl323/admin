## 常见错误及解决

### 错误 1: PM2 服务停止

**症状**: `pm2 status` 显示 `stopped`

**解决**:
```bash
pm2 start english-learning-backend
pm2 save
```

### 错误 2: 语法错误

**症状**: 日志显示 `SyntaxError` 或 `Unexpected token`

**原因**: 上传的代码有语法错误

**解决**:
```bash
# 检查最近的更改
cd /root/english-learning/backend
git status
git log -1

# 回滚到上一个版本
git reset --hard HEAD~1
pm2 restart english-learning-backend
```

### 错误 3: 数据库锁定

**症状**: 日志显示 `SQLITE_BUSY` 或 `database is locked`

**解决**:
```bash
cd /root/english-learning/backend
pm2 stop english-learning-backend
rm database.sqlite
pm2 start english-learning-backend
```

### 错误 4: 前端文件缺失

**症状**: PM2 正常,但页面显示错误

**检查**:
```bash
ls -lh /var/www/html/learning/
ls -lh /var/www/html/learning/index.html
```

**解决**:
```bash
# 使用 WinSCP 重新上传前端文件
# 或恢复备份
cp -r /var/www/html/learning-backup-* /var/www/html/learning
```
