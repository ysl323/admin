## 如果浏览器仍显示"获取验证码失败"

### 原因

问题是**浏览器缓存**了旧的前端代码。

### 解决方案（3步）

#### 1. 清除浏览器缓存

```
按 Ctrl + Shift + Delete
选择"缓存的图片和文件"
点击"清除数据"
```

#### 2. 强制刷新页面

```
按 Ctrl + F5
或 Ctrl + Shift + R
```

#### 3. 测试

访问: http://localhost:5173/register

应该看到验证码问题，如: **"5 + 3 = ?"**

## 调试方法

如果清除缓存后仍然失败:

### 1. 打开开发者工具

按 **F12**

### 2. 查看 Console 标签页

查找红色错误信息

### 3. 查看 Network 标签页

1. 刷新页面 (F5)
2. 找到 `captcha` 请求
3. 查看:
   - **状态码**: 应该是 200
   - **响应**: 应该有 JSON 数据

### 4. 预期响应

```json
{
  "success": true,
  "captchaId": "abc123...",
  "question": "5 + 3 = ?"
}
```
