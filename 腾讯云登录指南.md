# 腾讯云服务器登录指南

## 方法：通过腾讯云控制台登录服务器

1. 打开浏览器访问：https://console.cloud.tencent.com/
2. 登录你的腾讯云账号
3. 点击左上角"云服务器" → "实例"
4. 找到你的服务器实例
5. 点击"登录"按钮
6. 选择"标准登录"或"VNC登录"
7. 输入密码：Admin88868
8. 登录成功后，执行以下命令：

```bash
# 查看备份
ls -t /root/backup/ | head -3

# 恢复文件
cp -r /root/backup/dist-2026*/* /root/english-learning/frontend/dist/

# 重启Nginx
nginx -s reload

# 测试网站
curl http://47.97.185.117/
```
