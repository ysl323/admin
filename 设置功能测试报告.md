  id INTEGER PRIMARY KEY AUTOINCREMENT,
  user_id INTEGER NOT NULL,
  key VARCHAR(100) NOT NULL,
  value TEXT,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,

## ⚠️ 常见问题排查

### 问题1: 看不到设置按钮

**可能原因**: 浏览器缓存

**解决方案**:
```
1. 按 F12 打开开发者工具
2. Application → Storage → Clear site data
3. 刷新页面（Ctrl + F5）
```

### 问题2: 快捷键不生效

**可能原因**:
- 焦点在输入框内
- 浏览器快捷键冲突

**解决方案**:
```
1. 点击页面空白处，确保焦点不在输入框
2. 按F12查看控制台，应该看到"按键检测"日志
3. 尝试其他组合键
```

### 问题3: 快捷键设置未保存

**可能原因**: 后端服务未启动或数据库未创建

**验证命令**:
```bash
ssh root@47.97.185.117 "pm2 status"
ssh root@47.97.185.117 "sqlite3 /www/wwwroot/english-learning/backend/database.sqlite 'SELECT * FROM user_settings;'"
```
