# 学习模式功能部署说明

## 前提条件
- 服务器已安装 Node.js、npm、pm2、nginx
- 项目代码已推送到 Git 仓库

## 部署步骤

### 方法一: 通过腾讯云控制台部署(推荐)

1. 登录腾讯云控制台: https://console.cloud.tencent.com/
2. 进入云服务器 → 实例 → 点击"登录"
3. 登录密码: `Admin88868`
4. 登录后依次执行以下命令:

```bash
# 进入项目目录
cd /root/english-learning/my1

# 拉取最新代码
git pull origin master

# 执行部署脚本
chmod +x 服务器端部署脚本.sh
./服务器端部署脚本.sh
```

### 方法二: 手动执行部署步骤

如果自动化脚本失败,可以手动执行以下步骤:

```bash
# 1. 备份
cd /root/english-learning/my1
mkdir -p /root/backup-$(date +%Y%m%d_%H%M%S)
cp -r backend frontend /root/backup-$(date +%Y%m%d_%H%M%S)/

# 2. 拉取代码
git pull origin master

# 3. 数据库迁移
cd backend
sqlite3 database.sqlite <<EOF
DROP INDEX IF EXISTS unique_user_word_mastery;
CREATE UNIQUE INDEX IF NOT EXISTS unique_user_lesson_word_mastery
ON word_mastery (userId, lessonId, wordId);
EOF

# 4. 安装后端依赖
npm install --production

# 5. 初始化数据库
npm run db:init

# 6. 构建前端
cd ../frontend
npm install
npm run build

# 7. 重启后端
cd ..
pm2 restart english-backend || pm2 start backend/src/index.js --name english-backend

# 8. 检查状态
pm2 status
pm2 logs english-backend --lines 20

# 9. 重启nginx
nginx -s reload
```

## 回滚方案

如果部署后出现问题,可以快速回滚:

```bash
# 找到最近的备份
ls -lt /root/backup-* | head -1

# 恢复备份
BACKUP_DIR=$(ls -dt /root/backup-* | head -1)
cp -r $BACKUP_DIR/backend/* /root/english-learning/my1/backend/
cp -r $BACKUP_DIR/frontend/* /root/english-learning/my1/frontend/
cp $BACKUP_DIR/database.sqlite /root/english-learning/my1/backend/

# 重启服务
pm2 restart english-backend
nginx -s reload
```

## 性能优化建议

1. 启用PM2集群模式:
   ```bash
   pm2 delete english-backend
   pm2 start backend/src/index.js --name english-backend -i max
   pm2 save
   ```

2. 配置Nginx缓存(可选)

3. 启用Gzip压缩(通常已配置)

4. 定期清理PM2日志:
   ```bash
   pm2 install pm2-logrotate
   pm2 set pm2-logrotate:max_size 10M
   pm2 set pm2-logrotate:retain 7
   ```
