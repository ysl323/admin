## 🎯 功能概述

实现了多单词短语的智能输入体验：

### 核心功能
1. **智能输入框**: 根据单词数量自动显示对应数量的输入框
2. **逐词验证**: 每输入一个单词按空格立即验证
3. **错误处理**: 输入错误时清空、播放音频、等待重新输入
4. **自动聚焦**: 验证正确后自动聚焦到下一个输入框
5. **视觉反馈**: 错误时显示红色下划线和抖动动画

## 🔧 技术实现

### 核心代码结构

#### 1. 数据结构
```javascript
const wordParts = ref([]);        // ['hello', 'world']
const wordInputs = ref([]);       // ['', '']
const wordInputRefs = ref([]);    // [ref1, ref2]
const currentWordIndex = ref(0);  // 0, 1, 2...
const wordErrors = ref([]);       // [false, false]
```

#### 2. 关键函数
```javascript
// 初始化输入框
initWordInputs()

// 处理单词输入
handleWordInput(index)

// 处理单词完成（按空格）
handleWordComplete(index)

// 提交多单词答案
handleMultiWordSubmit()
```

#### 3. UI 组件
```vue
<!-- 单个单词 -->
<input v-if="wordParts.length === 1" ... />

<!-- 多个单词 -->
<div v-else class="multi-word-input">
  <div v-for="(part, index) in wordParts">
    <input
      :disabled="currentWordIndex !== index"
      :class="{ 'error': wordErrors[index] }"
      @keydown.space.prevent="handleWordComplete(index)"
    />
  </div>
</div>
```
