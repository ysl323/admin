## 关于上传课程失败

### 检查上传功能
上传课程需要：
1. **管理员权限** - 必须以管理员身份登录
2. **正确的JSON格式** - 数据格式必须符合要求

### 支持的导入格式

#### 格式1：简化格式（新概念英语）
```json
[
  {"lesson": 1, "question": 1, "english": "hello", "chinese": "你好"},
  {"lesson": 1, "question": 2, "english": "world", "chinese": "世界"}
]
```

#### 格式2：标准格式
```json
{
  "name": "新分类",
  "lessons": [
    {
      "lesson": 1,
      "words": [
        {"question": 1, "english": "hello", "chinese": "你好"}
      ]
    }
  ]
}
```

### 常见错误
1. **401 Unauthorized** - 未登录或权限不足
2. **400 Bad Request** - JSON格式错误
3. **500 Internal Server Error** - 服务器错误
