**检查方法**: 打开浏览器开发者工具 (F12)，查看 Console 和 Network 标签

## 常见错误和解决方案

### 错误1: "获取验证码失败"

**原因**: 
- 后端未运行
- 网络请求失败
- API返回错误

**解决**:
1. 检查后端是否运行: `curl http://localhost:3000/api/captcha`
2. 查看浏览器 Network 标签的实际错误
3. 查看后端日志: `backend/logs/combined.log`

### 错误2: CORS 错误

**原因**: 跨域请求被阻止

**解决**:
1. 检查 `backend/.env` 中的 `CORS_ORIGIN`
2. 确保后端 CORS 配置正确
3. 重启后端服务器

### 错误3: 404 Not Found

**原因**: API路径不正确

**解决**:
1. 确认后端路由: `/api/captcha`
2. 确认前端请求: `api.get('/captcha')`
3. 确认代理配置: `proxy: { '/api': { target: 'http://localhost:3000' } }`

### 错误4: 连接被拒绝

**原因**: 后端服务器未运行

**解决**:
```bash
cd my1/backend
npm start
```
