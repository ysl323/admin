## 关于上传课程失败

如果上传课程仍然失败，请检查：

### 1. 文件格式
- 必须是 JSON 格式或 UTF-8 编码的 TXT 文件
- 不能使用其他编码格式

### 2. 数据格式
JSON 数据必须是数组格式，例如：
```json
[
  {
    "lesson": 1,
    "question": 1,
    "english": "Hello",
    "chinese": "你好"
  }
]
```

必填字段：
- `question`: 序号
- `english`: 英文
- `chinese`: 中文

可选字段：
- `lesson`: 课时号（如果有则按课时分组）

### 3. 查看错误信息
上传失败时，查看浏览器控制台或后端日志中的详细错误信息。
