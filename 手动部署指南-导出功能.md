## 常见问题

### Q1: 上传后看不到按钮
A: 清除浏览器缓存 (Ctrl + F5)

### Q2: 导出按钮点击没反应
A: 检查浏览器控制台 (F12),查看错误信息

### Q3: PM2 重启失败
A:
```bash
pm2 stop english-learning-backend
pm2 delete english-learning-backend
cd /root/english-learning/backend
pm2 start index.js --name english-learning-backend
pm2 save
```

### Q4: 导出返回 404
A: 确认后端文件已正确上传,重启 PM2 服务

### Q5: 导入功能失败
A: 检查 JSON 格式是否正确,查看 PM2 日志:
```bash
pm2 logs english-learning-backend --lines 50
```

## 技术细节

### 更新的文件
1. `AdminService.js` - 添加了导出方法
2. `admin.js` - 添加了导出 API 路由
3. `ContentManagement.vue` - 添加了导出按钮和处理函数
4. `admin.js` (前端服务) - 添加了导出 API 调用

### 新增的 API 端点
- `GET /api/admin/export/all` - 导出所有数据
- `GET /api/admin/export/category/:id` - 导出指定分类
- `GET /api/admin/export/lesson/:id` - 导出指定课程

### 导出数据格式
```json
{
  "success": true,
  "data": [
    {
      "name": "分类名称",
      "lessons": [
        {
          "lesson": 1,
          "words": [
            {

              "english": "hello",
              "chinese": "你好"
            }
          ]
        }
      ]
    }
  ],
  "stats": {
    "categories": 1,
    "lessons": 1,
    "words": 1
  }
}
```
